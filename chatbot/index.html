CORS Connection Issue

Browsers restrict direct localhost fetches. Try these fixes:

Option A: CORS Proxy (Running)

We started a local proxy server on port 8001 that adds missing headers. Set endpoint above to http://127.0.0.1:8001.

Option B: Browser Extension

Install Allow CORS: Access-Control-Allow-Origin and activate it.

Option C: Disable Web Security (Mac)
open -n -a /Applications/Google\ Chrome.app --args --user-data-dir="/tmp/chrome_dev_test" --disable-web-security
